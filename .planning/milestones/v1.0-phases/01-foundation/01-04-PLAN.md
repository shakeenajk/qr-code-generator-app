Create `src/components/Logo.astro` — the QRCraft SVG logo. The letter "Q" is built from a 7×7 grid of small square dots (10×10 px each, 2px gap → 12px pitch), mimicking QR code visual language. All dots use `fill="currentColor"` so the parent's text color drives the fill.

The "Q" dot pattern on a 7×7 grid (column 0–6, row 0–6), using (col*12, row*12) for x/y:
- Top bar: cols 2–4, row 0
- Left side: col 1, rows 1–5; col 2, rows 1–2 (corner fill)
- Right side: col 5, rows 1–5; col 4, rows 1–2 (corner fill)
- Bottom bar: cols 2–4, row 5
- Tail: col 4, row 5 (already in bottom bar); col 5, row 6 (the Q's tail descender)
- Inner fill (letter body): cols 2–4, rows 2–4 are EMPTY (open center of Q)

```astro
---
// src/components/Logo.astro
interface Props {
  class?: string;
  size?: number;

---

  xmlns="http://www.w3.org/2000/svg"
  viewBox="0 0 84 84"
  width={size}
  height={size}
  aria-label="QRCraft logo"
  role="img"
  class={`text-[#2563EB] ${className}`}
>
```
