    **Setup (required first):**
    1. Go to https://dashboard.clerk.com and create an application (or use existing)
    2. In Clerk Dashboard: enable Google and GitHub OAuth providers
    3. Set `PUBLIC_CLERK_PUBLISHABLE_KEY` and `CLERK_SECRET_KEY` in .env.local
    4. Run `npm run dev`

    **AUTH-01: Email/password sign-up**
    1. Visit http://localhost:4321/signup
    2. Confirm: full site header + footer visible, Clerk SignUp form renders with blue button
    3. Sign up with a test email + password
    4. Confirm: redirected to http://localhost:4321/dashboard after sign-up
    5. Confirm: dashboard shows "My QR Codes" empty state

    **AUTH-02: Email/password sign-in**
    1. Sign out first (via dropdown → Sign Out)
    2. Visit http://localhost:4321/login
    3. Sign in with the credentials from AUTH-01
    4. Confirm: redirected to /dashboard

    **AUTH-03: OAuth buttons visible**
    1. Visit http://localhost:4321/login
    2. Confirm: Google and GitHub OAuth buttons are visible in the Clerk SignIn card
    3. (Optional) Click "Sign in with Google" or "Sign in with GitHub" and complete the OAuth flow

    **AUTH-04: Session persistence**
    1. Sign in and navigate to /dashboard
    2. Refresh the browser (Cmd+R / Ctrl+R)
    3. Confirm: you remain on /dashboard (not redirected to /login)

    **AUTH-05: Sign out**
    1. While signed in, click the avatar in the header
    2. Confirm: dropdown shows "My Dashboard", "Account Settings", "Sign Out"
    3. Click "Sign Out"
    4. Confirm: redirected to http://localhost:4321/

    **Middleware redirect**
    1. Sign out completely
    2. Visit http://localhost:4321/dashboard directly
    3. Confirm: redirected to /login

    **Header states**
    1. While signed out: confirm header shows "Sign In" button (not avatar)
    2. While signed in: confirm header shows avatar with profile photo or initials + name + chevron

    **Mobile layout (optional)**
    1. Open DevTools, set to mobile viewport (375px width)
    2. Visit /dashboard
    3. Confirm: bottom tab bar is visible with 3 icons, sidebar is hidden
