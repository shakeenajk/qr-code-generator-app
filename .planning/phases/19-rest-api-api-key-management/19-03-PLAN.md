       **State:**
       - `keys`: array of `{ id, name, keyPrefix, usageCount, lastUsedAt, revokedAt, createdAt }`

       - "Create API Key" button at top (primary blue button)
       - Clicking opens a form/modal inline with a text input for key name and "Create" button
       - On submit: POST /api/dashboard/api-keys with `{ name }`

       - A "Done" button below clears `newKeyRaw` and refreshes the key list
       - The raw key card disappears permanently once "Done" is clicked

       **Revoke flow:**
       - Click "Revoke" button on a key row
       - Show a confirm dialog (simple `window.confirm('Revoke key "keyPrefix..."? This cannot be undone.')`)
       - On confirm: DELETE /api/dashboard/api-keys/{id}
       - On success: `toast.success('API key revoked')`, refresh key list
       - On error: `toast.error('Failed to revoke key')`

       **Empty state:**
       - If no keys exist: "No API keys yet. Create one to start using the QRCraft API."

       - Monospace font for key prefixes: `font-mono`
