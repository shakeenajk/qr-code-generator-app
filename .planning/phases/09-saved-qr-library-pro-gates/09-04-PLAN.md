Create `src/components/dashboard/QRLibrary.tsx` as a React component (no `client:only` needed here — the Astro page will add that). This is the complete library view.

**State:**
```typescript
const [qrCodes, setQrCodes] = useState<SavedQR[]>([]);
const [loading, setLoading] = useState(true);
const [viewMode, setViewMode] = useState<'grid' | 'list'>('grid');  // default grid per CONTEXT
const [deletingId, setDeletingId] = useState<string | null>(null);   // tracks which card is pending delete confirm
```

**Data fetch on mount:**
```typescript
useEffect(() => {
  fetch('/api/qr/list')
    .then(r => r.json())
    .then(data => { setQrCodes(data); setLoading(false); })
    .catch(() => setLoading(false));
}, []);
```

**handleDelete(id: string):**
- Set deletingId = id (shows inline confirmation on the card)
- When user clicks "Confirm Delete": call `fetch('/api/qr/[id]', { method: 'DELETE' })`
- On success: setQrCodes(prev => prev.filter(q => q.id !== id)); toast('QR code deleted'); setDeletingId(null)
- On error: toast.error('Delete failed'); setDeletingId(null)

**Inline delete confirmation (not a modal — per Claude's discretion):**
When `deletingId === card.id`, replace Edit + Delete buttons with "Are you sure?" text + "Yes, delete" (red) and "Cancel" buttons on that card only.

**handleEdit(id: string):**
```typescript
window.location.href = `/?edit=${id}`;
```

**Render structure:**

Loading state: spinner or "Loading..." text, centered.

Empty state (qrCodes.length === 0 after loading):
- SVG illustration (simple — can use an inline SVG of a QR code with a dashed border, or an emoji-based fallback)
- `<h2>No QR codes yet</h2>`
- `<p>Save a generated QR code to build your library.</p>`
- `<a href="/">Go to Generator</a>` — styled as a primary button (bg-indigo-600 text-white rounded px-4 py-2)

Library header (shown when qrCodes.length > 0):
- "My QR Codes" heading (h2, text-xl font-semibold)
- Grid/List toggle buttons on the right: two icon buttons (Grid2x2 and List from lucide-react), active state highlighted

**Grid view** (`viewMode === 'grid'`):
`grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4`

- White/dark card with rounded-lg shadow-sm border
- Top: thumbnail image (if thumbnailData is set, render `<img src={qr.thumbnailData} alt={qr.name} className="w-full aspect-square object-contain rounded-t-lg bg-white p-4" />`; if null, render a grey placeholder square with a QR code icon)

**List view** (`viewMode === 'list'`):
`flex flex-col gap-2`

All UI: Tailwind v4, dark mode variants (`dark:bg-gray-800`, `dark:text-white`, `dark:border-gray-700`). Consistent with existing dashboard components.
